== Description ==

Built on wordpress, TriMS is a complete makeover of the Wordpress Dashboard; Developed by Triangle Mena.

== Installation ==

1. Upload `TriMS` to the `/assets/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check the sharing buttons in the default posts page and all the custom posts pages
